# Angular CLI Commands

## Adding a new root module

1. Run the following command:

```bash
ng g module modules/[name] --module=app --routing=true --project=[project]
```

2. Update `app-routing.module.ts` with a new route

```typescript
  {
    path: '[name]',
    loadChildren: () => import('./[name]/[name].module').then((m) => m.[Name]Module),
  },
```

## Adding a new feature to a module

```bash
ng g module modules/[module]/features/[name] --module=[module] --routing=true --route=[name] --project=[project]
```

## Adding a component

### Smart component

```bash
ng g module modules/[module]/features/[feature]/[feature]-[name] --project=[project]
ng g component modules/[module]/features/[feature]/[feature]-[name] --project=[project]
```

### Dumb component

```bash
ng g module modules/[module]/components/[name] --project=[project]
ng g component modules/[module]/components/[name] --project=[project]
```
