# My common angular commands

## Adding a new root module

1. Run the following command:

<InteractiveCode
  client:load
  lang="bash"
  code="ng g module modules/[name] --module=app --routing=true --project=[project]"
  tokens={[
    { key: "[name]", label: "Name" },
    { key: "[project]", label: "Project" },
  ]}
/>

2. Update `app-routing.module.ts` with a new route

<InteractiveCode
  client:load
  lang="typescript"
  code={`  {
    path: '[name]',
    loadChildren: () => import('./[name]/[name].module').then((m) => m.[Name]Module),
  },`}
  tokens={[
    { key: "[name]", label: "Name (Path)" },
    { key: "[Name]", label: "Name (Module)" },
  ]}
/>

## Adding a new feature to a module

<InteractiveCode
  client:load
  lang="bash"
  code={`ng g module modules/[module]/features/[name] --module=[module] --routing=true --route=[name] --project=[project]`}
  tokens={[
    { key: "[module]", label: "Module" },
    { key: "[name]", label: "Name" },
    { key: "[project]", label: "Project" },
  ]}
/>

## Adding a component

### Smart component

<InteractiveCode
  client:load
  lang="bash"
  code={`ng g module modules/[module]/features/[feature]/[feature]-[name] --project=[project]
ng g component modules/[module]/features/[feature]/[feature]-[name] --project=[project]`}
  tokens={[
    { key: "[module]", label: "Module" },
    { key: "[feature]", label: "Feature" },

  ]}
/>
